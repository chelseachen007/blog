>```
> Given a list of integers nums, return the number of pairs i < j such that nums[i] > nums[j] * 3.
> 
> Constraints
>  
> n ≤ 100,000 where n is the length of nums
>  Example 1
> Input
>  nums = [7, 1, 2]
>Output
>2
> Explanation
>We have the pairs (7, 1) and (7, 2)
> ```
